# Masuno
Sistema de reservas para salón de belleza.

## Estructura del proyecto

- **public/**: Punto de entrada y recursos públicos.
- **app/controllers/**: Controladores.
- **app/models/**: Modelos.
- **app/views/**: Vistas.
- **core/**: Clases base (Router, Controller, Database).
- **config/**: Configuración de la base de datos.

## Instalación

1. Clonar el repositorio.
2. Ejecutar `composer install`.
3. Configurar `config/database.php`.
4. Crear la base de datos `masuno`.
5. Iniciar servidor local (XAMPP, WAMP, etc.).

## Uso

Visitar http://localhost/masuno/public
